## 글로벌 전략

### 타겟 시장 선정 근거

Sensor Tower 크롤링 (51개 데이터셋, 18개 컬럼 직접 설계) + 자막 파일 소비 문화 교차 분석.

| 국가 | Android 점유율 | 자막 파일 문화 | OTT 환경 |
|------|---------------|---------------|----------|
| 러시아 | 70% | 강함 | 서방 OTT 제재 |
| 브라질 | 85% | 강함 (포르투갈어 자막 커뮤니티) | 유료 구독 거부감 |
| 인도네시아 | 90%+ | 강함 (한드·일애니 자막) | 해적판 유통 큼 |
| 베트남 | 90%+ | 강함 (한드·영미권 자막) | 단속 부재 |

**공통**: Android 강세 + 자막 파일 직접 소유 습관 + 자막 커뮤니티 활성

### 마케팅 전략

- **한국 + 미국**: 개인 바이럴 최소한
- **나머지 4개국**: ASO 자연 유입 의존
- 마케팅 이미지 비언어화 (피처 그래픽 / 스크린샷)

## 다국어 텍스트 설계 원칙

**한국어 (원본)**: 토스 톤 — 구어체, 부드러움
**외국어**: 설명체 통일 — 감성보다 이해도 우선

### 검수 프로세스

- Claude Opus 4.7로 초벌 번역
- GPT-5 + Gemini 교차 검수
- 의견 갈리면 의도 분명한 건 유지, 단순 어색한 어휘만 교체
- 7군데 최종 수정

### 적용 범위 (15문장 × 6언어 = 90 항목)

- 온보딩 4화면 텍스트 (12 문장)
- 홈 화면 안내 (2 문장)
- 권한 안내 화면 (3 문장)
- 오버레이 자막 영역 안내 (2 문장)
- 시간 이동 다이얼로그 (7 항목)
- 입력 에러 메시지 (3 문장)
- 핸들 / 메뉴 아이콘 ContentDescription (6 항목)
- 파일 처리 토스트 (4 문장)
- 삭제 확인 다이얼로그 (3 항목)
